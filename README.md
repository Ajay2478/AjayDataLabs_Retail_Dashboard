## Overview
This interactive dashboard provides real-time insights into retail sales performance, allowing stakeholders to track KPIs, analyze regional trends, and filter data by category.

## Features
- **KPI Tracking:** Real-time calculation of Total Revenue and Profit Margins.
- **Interactive Visualizations:** Dynamic bar charts and time-series trends using Plotly.
- **Multiselect Filtering:** Custom views for different Regions and Product Categories.
- **Automated Data Pipeline:** Built-in script for synthetic data generation.

## Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **Libraries:** Pandas, Plotly, NumPy
- **Deployment:** Streamlit Cloud
